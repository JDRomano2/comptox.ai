About ComptoxAI

Computational toxicology is the branch of toxicology that uses computational modeling to predict toxic interactions in biological systems and explain mechanisms of interactions that are already known.

ComptoxAI is a toolkit designed to enable AI and data science research in computational toxicology. Like many disciplines in biomedicine, computational toxicology is experiencing a deluge of valuable data being generated in a wide array of modalities and levels of application. However, the tools for interacting with these data are - at best - inconsistent, outdated, and fail to leverage state-of-the-art computational methods. We intend for ComptoxAI to change this.

Main components of ComptoxAI

Toxicology graph database

ComptoxAIâ€™s graph database is meant to completely replace traditional relational databases used for computational toxicology data. Rather than storing data in tables - with keys linking rows of one table to rows of another - a graph database stores entities as nodes and the relationships that join entities as edges, resulting in a network-like data structure known to computer scientists as a graph.

Graph database algorithms

One of the key advantages of representing data as a graph is that it makes it almost trivial to implement algorithms that highlight important characteristics in networks. These include:

  • Shortest path finding between two nodes
  • Community detection
  • Page rank; for identifying the most influential nodes (made famous by Google)

All of these are (or soon will be) implemented natively in ComptoxAI, and can be as simple as running a single line of Python code.

ComptoxAI's ontology

An ontology is a certain type of knowledge representation that makes it possible to define concepts in a way that enables automated reasoning (i.e., in a way that computers can "understand"). It does this by linking objects arranged in a hierarchy via descriptive non-hierarchical relationships. Ontologies are a crucial component of modern artificial intelligence ecosystems, and nicely complement "knowledge-free" AI methods, like statistical computing and machine learning.

ComptoxAI includes a complete ontology that gives explicit restrictions to the data stored in the graph database.
